Modelo de Caja CSS

El modelo de caja en CSS describe cómo se representan y dimensionan los elementos en una página web. Cada elemento HTML es tratado como una caja rectangular con cuatro áreas clave: contenido, padding, borde y margen.

  • El contenido es el área interna que contiene el texto o los elementos hijos.
  • El padding es el espacio entre el contenido y el borde.
  • El borde rodea el contenido y el padding, definiendo el límite visual del elemento.
  • El margen es el espacio exterior al borde, que separa un elemento de otros en la página.

Las propiedades CSS como width, height, padding, border y margin controlan las dimensiones y apariencia de estas áreas. Por ejemplo, width y height definen las dimensiones del contenido, mientras que padding agrega espacio interno y margin agrega espacio externo. Estas propiedades permiten un control preciso del diseño y la apariencia de los elementos en una página web, facilitando la creación de diseños flexibles y atractivos.

Propiedades del Modelo de Caja

1. Ancho (width) y Alto (height): Establecen las dimensiones de la caja.

2. Márgenes (margin): Definen el espacio externo alrededor de la caja.

3. Relleno (padding): Determina el espacio interno dentro de la caja.

4. Borde (border): Define el borde alrededor de la caja.
